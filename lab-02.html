9.b. What does the following code chunk do? Run it and analyze the result and articulate in words what group_by() does. Also comment on how it’s different from the arrange() in part (a)?

df |>
  group_by(var_2)

group_by(var_2) groups the data by the values in var_2, so that future operations will be performed separately for each group. Unlike arrange(), it does not reorder the rows of the data frame; it simply adds grouping information to the dataset.

9.c. What does the following code chunk do? Run it and analyze the result and articulate in words what the pipeline does.

df |>
  group_by(var_2) |>
  summarize(mean_var_1 = mean(var_1))
# A tibble: 2 × 2
  var_2  mean_var_1
  <chr>       <dbl>
1 Burger       35  
2 Pizza        26.7

This chunk of code first groups the data by var_2 using group_by(). It then uses summarize() to calculate the mean value of var_1 within each group. As a result, the output shows the average value of var_1 for each category in var_2, producing one row per group.

df |>

9.f. What do the following pipelines do? Run both and analyze their results and articulate in words what each pipeline does. How are the outputs of the two pipelines different?

df |>
  group_by(var_2, var_3) |>
  summarize(mean_var_1 = mean(var_1), .groups = "drop")
# A tibble: 4 × 3
  var_2  var_3  mean_var_1
  <chr>  <chr>       <dbl>
1 Burger Apple          20
2 Burger Banana         50
3 Pizza  Apple          10
4 Pizza  Pear           35
df |>
  group_by(var_2, var_3) |>
  mutate(mean_var_1 = mean(var_1))
# A tibble: 5 × 4
# Groups:   var_2, var_3 [4]
  var_1 var_2  var_3  mean_var_1
  <dbl> <chr>  <chr>       <dbl>
1    10 Pizza  Apple          10
2    20 Burger Apple          20
3    30 Pizza  Pear           35
4    40 Pizza  Pear           35
5    50 Burger Banana         50

The first pipeline groups the data by var_2 and var_3 and uses summarize() to calculate the mean of var_1 for each group, resulting in one row per group. The second pipeline also groups the data by var_2 and var_3, but uses mutate() to add the group mean of var_1 as a new column while keeping all original rows. Therefore, the outputs differ because summarize() reduces the number of rows while mutate() preserves the original dataset and adds additional information.
